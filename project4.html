Personal_Trainee

Project Screenshot

Personal Trainee AI is an intelligent virtual assistant designed to guide users in learning technical skills and preparing for job placements. The system uses AI to create personalized learning paths, offer tool-based training (e.g., penetration testing), and conduct mock interviews with real-time emotional analysis to improve performance and readiness for real-world roles.

Key Features:

• Personalized skill roadmap: Based on user's interests or goals.

• Step-by-step learning paths: With tools, tutorials, and practice tasks.

• Mock interviews: With AI-driven emotional feedback (using SER model).

• Resume analysis: And improvement suggestions.

• Job role recommendations: With missing skill detection.

• Platform integration: With LinkedIn and Naukri for job search.

Tech Stack:
Flutter, Firebase, Python, TensorFlow/Keras, OpenAI API, Web Scraping (for job listings), FFmpeg (for voice processing), NLP + CNN (for mock interview SER)
